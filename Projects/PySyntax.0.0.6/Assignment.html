Operatori Di Assegnazione

Assegnazione semplice

Sintassi 

variabile = valore da assegnare

Esempio

Codice 
X = 5
print ( X )
Output 
5

Esempio

Codice 
X = 5 + 5
print ( X )
Output 
10

Esempio

Codice 
X = 5 * 'a'
print ( X )
Output 
aaaaa

Esempio

Codice 
X = 5
Y = X
print ( Y )
Output 
5

Esempio

Codice 
A = 5
B = 7
X = A * B
print ( X )
Output 
35
