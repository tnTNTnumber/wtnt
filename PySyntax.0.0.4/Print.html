Funzioni

Print

Sintassi 

print ( oggetti , sep = separatore , end = fine , file = file , flush = flush )
Parametri: 
oggetto: può essere uno o più (nel caso siano più di uno devono essere separati da una virgola) e può essere un numero, una stringa oppure una variabile. Valore predefinito: None
separatore: separa gli oggetti tra loro. Valore predefinito: ' '
fine: è stampato alla fine. Valore predefinito: '\n'
file: dice dove deve essere stampato il contenuto (deve essere un oggetto con metodo write (string)). Valore predefinito: sys.stdout
flush: se True, il flusso viene scaricato forzatamente. Valore predefinito: False

Codice 
print ( 'ciao' )
Output 
ciao

5
