Variabili

Dizionari

Sintassi 

{ chiave 0: elemento 0, chiave 1: elemento 1, chiave 2: elemento 2, chiave 3: elemento 3, ... }
Spiegazione: 
In Python, le serie, sono elenchi ordinati di elementi e sostanzialmente sono simili alle liste.
Chiave: è una stringa, od un numero, che viene assegnato come nome dell'elemento.

Esempio

Codice 
print ( { 'a': 1, 2: 2, 'ciao': 3 } )
Output 
{'a': 1, 2: 2, 'ciao': 3}

Esempio

Codice 
print ( { 'Gatto': 'Europeo' } )
Output 
{'Gatto': 'Europeo'}

Esempio

Codice 
print ( { 'a': 'gelato', 'b': 'festa', 'c': 'passeggiata' } )
Output 
{'a': 'gelato', 'b': 'festa', 'c': 'passeggiata'}

Esempio

Codice 
X = 'Bistecca'
print ( { 'Contenuto 1': 'Ciao', 'Contenuto 2': 5, 'Contenuto 3': True, 'Contenuto 4': X } )
Output 
{'Contenuto 1': 'Ciao', 'Contenuto 2': 5, 'Contenuto 3': True, 'Contenuto 4': 'Bistecca'}
