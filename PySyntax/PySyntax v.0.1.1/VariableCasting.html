Variabili

Tipologia

Sintassi 

variabile = classe ( contenuto )
Spiegazione: 
In Python, la dichiarazione della tipologia di una variabile, si può eseguire mediante la dichiarazione di una variabile con l'aggiunta di una funzione che imposti la tipologia del contenuto.

Esempio

Codice 
X = int ( 5 )
Y = str ( 5 )
Z = float ( 5 )
print ( 'In questo caso, la variabile X, contiene un numero intero.' )
print ( 'X continene:', X )
print ( )
print ( 'In questo caso, la variabile Y, contiene una stringa.' )
print ( 'Y continene:', Y )
print ( )
print ( 'In questo caso, la variabile Z, contiene un numero decimale.' )
print ( 'Z continene:', Z )
Output 
In questo caso, la variabile X, contiene un numero intero.
X continene: 5

In questo caso, la variabile Y, contiene una stringa.
Y continene: 5

In questo caso, la variabile Z, contiene un numero decimale.
Z continene: 5.0
