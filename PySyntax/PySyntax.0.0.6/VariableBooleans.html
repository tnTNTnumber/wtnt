Booleani

Sintassi 

booleano
Spiegazione: 
In Python, i booleani, sono dei valori binari che possono essere True o False:
True sta per "Vero", e convertito in numero corrisponde ad 1.
False sta per "False", e convertito in numero corrisponde a 0.

Esempio

Codice 
print ( True )
print ( int ( True ) )
print ( )
print ( False )
print ( int ( False ) )
Output 
True
1

False
0
