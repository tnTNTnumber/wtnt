Variabili

Leggere i Dizionari

Sintassi 

dizionario [ chiave ]
Spiegazione: 
In Python, per leggere i dizionari, dopo il dizionario (che di solito è indicato con una variabile) si scrive tra [ ] la chiave dell'oggetto da leggere.
Chiave: è una stringa, od un numero, che viene assegnato come nome dell'elemento, funziona similmente al numero tra [ ] negli liste, nelle ennuple, e nelle Serie.

Esempio

Codice 
print ( 'La chiave 1 corrisponde al numero 2:' )
print ( { 1: 4, 2: 5, 3: 6 } [ 1 ] )
print ( )
print ( 'La chiave "b" corrisponde al numero 2:' )
print ( { 'a': 1, 'b': 2, 'c': 3 } [ 'b' ] )
print ( )
print ( 'La chiave "None" corrisponde al numero 3:' )
print ( { True: 1, False: 2, None: 3 } [ None ] )
Output 
La chiave 1 corrisponde al numero 2:
4

La chiave "b" corrisponde al numero 2:
2

La chiave "None" corrisponde al numero 3:
3

Esempio

Codice 
X = 'Bistecca'
print ( 'Alla chiave "Primo" corrisponde "Bistecca":' )
print ( { 'Primo': X, 'Secondo': 'Insalata', 'Bevanda': 'Acqua' } [ 'Primo' ] )
Output 
Alla chiave "Primo" corrisponde "Bistecca":
Bistecca

Esempio

Codice 
X = 'Bistecca'
A = { 'Primo': X, 'Secondo': 'Insalata', 'Bevanda': 'Acqua' }
print ( 'Un dizionario è comune che venga assegnato ad una variabile.' )
print ( 'In questo caso è assegnato alla variabile "A", perciò per chiamarlo bisogna scrivere la variabile "A" invece che il dizionario:' )
print ( )
print ( 'Alla chiave "Bevanda" corrisponde "Acqua":' )
print ( A [ 'Bevanda' ] )
Output 
Un dizionario è comune che venga assegnato ad una variabile.
In questo caso è assegnato alla variabile "A", perciò per chiamarlo bisogna scrivere la variabile "A" invece che il dizionario:

Alla chiave "Bevanda" corrisponde "Acqua":
Acqua

X = 'Tre'

Esempio

Codice 
print ( 'Se un dizionario contiene altri dizionari si può accedere ad i loro contenuti in questo modo:' )
print ( )
print ( { 'a': { 'a1': 1, 'a2': 2 }, 'b': { 'b1': 3, 'b2': 4 } } [ 'b' ] [ 'b1' ] )
print ( 'In questo caso nelle prime [] vi è contenuto "\'b\'", questo indica che si debba leggere il contenuto corrispondente alla chiave "\'b\'":' )
print ( 'Ovvero "{ \'b1\': 3, \'b2\': 4 }".' )
print ( 'A questo punto bisogna immaginarsi come se ci fosse scritto "{ \'b1\': 3, \'b2\': 4 } [ \'b1\' ]", ovvero le seconde [] indicano la posizione nel dizionario andato a leggere.' )
print ( )
print ( 'In altre parole le chiavi sono ripettivamente \'b\' e \'b1\', quindi bisogna andare a leggere prima nella chiave "\'b\'", ed all\'interno della chiave "\'b\'", la chiave "\'b1\'".' )
Output 
Se un dizionario contiene altri dizionari si può accedere ad i loro contenuti in questo modo:

3
In questo caso nelle prime [] vi è contenuto "'b'", questo indica che si debba leggere il contenuto corrispondente alla chiave "'b'":
Ovvero "{ 'b1': 3, 'b2': 4 }".
A questo punto bisogna immaginarsi come se ci fosse scritto "{ 'b1': 3, 'b2': 4 } [ 'b1' ]", ovvero le seconde [] indicano la posizione nel dizionario andato a leggere.

In altre parole le chiavi sono ripettivamente 'b' e 'b1', quindi bisogna andare a leggere prima nella chiave "'b'", ed all'interno della chiave "'b'", la chiave "'b1'".
