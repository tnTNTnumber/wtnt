Instruzioni Condizionali

If Else

Sintassi 

qui può esserci del codice ...
if condizione :
valutazione della condizione
	se la condizione è vera va qui
else :
	se la condizione è falsa va qui
qui può continuare il codice ...

Esempio

Codice 
X = 5
if X == 5 :
	print('La condizione è vera e va qui')
else :
	print('La condizione è vera ed è già andata sopra, quindi non andrà qui')
print('Qui può continuare il codice ...')
Output 
La condizione è vera e va qui
Qui può continuare il codice ...

Esempio

Codice 
X = 3
if X == 5 :
	print('La condizione è falsa, quindi non andrà qui')
else :
	print('La condizione è falsa e non è andata di sopra, infatti va qui')
print('Qui può continuare il codice ...')
Output 
La condizione è falsa e non è andata di sopra, infatti va qui
Qui può continuare il codice ...
