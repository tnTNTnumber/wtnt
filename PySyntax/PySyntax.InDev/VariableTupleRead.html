Leggere le Ennuple

Sintassi 

ennupla [ posizione dell'elemento da leggere ]
Spiegazione: 
In Python, per leggere le ennuple, dopo la ennupla (che di solito è indicata con una variabile) si scrive tra [ ] la posizione dell'oggetto da leggere (ricordandosi che nelle ennuple la prima posizione è 0).

Esempio

Codice 
print ( 'In posizione 1 c\'è il numero 2:' )
print ( ( 1, 2, 3 ) [ 1 ] )
print ( )
print ( 'In posizione 0 c\'è la lettera "a":' )
print ( ( 'a', 'b', 'c' ) [ 0 ] )
print ( )
print ( 'In posizione 2 c\'è "None":' )
print ( ( True, False, None ) [ 2 ] )
Output 
In posizione 1 c'è il numero 2:
2

In posizione 0 c'è la lettera "a":
a

In posizione 2 c'è "None":
None

Esempio

Codice 
X = 'Bistecca'
print ( 'In posizione 3 c\'è "Bistecca":' )
print ( ( 'Ciao', 5, True, X ) [ 3 ] )
Output 
In posizione 3 c'è "Bistecca":
Bistecca

Esempio

Codice 
X = 'Bistecca'
A = ( 'Ciao', 5, True, X )
print ( 'Una ennupla è comune che venga assegnata ad una variabile.' )
print ( 'In questo caso è assegnata alla variabile "A", perciò per chiamarla bisogna scrivere la variabile "A" invece che la ennupla:' )
print ( )
print ( 'In posizione 0 c\'è "Ciao":' )
print ( A [ 0 ] )
Output 
Una ennupla è comune che venga assegnata ad una variabile.
In questo caso è assegnata alla variabile "A", perciò per chiamarla bisogna scrivere la variabile "A" invece che la ennupla:

In posizione 0 c'è "Ciao":
Ciao

Esempio

Codice 
print ( ( ( 1, 2 ), ( ( 1, 2 ), 2, 3 ) ) [ 1 ] )
Output 
((1, 2), 2, 3)

Esempio

Codice 
print ( 'Se una ennupla contiene altre ennuple si può accedere ad i loro contenuti in questo modo:' )
print ( )
print ( ( ( 1, 2 ), ( 3, 4 ) ) [ 1 ] [ 0 ] )
print ( 'In questo caso nelle prime [] vi è contenuto "1", questo indica che si debba leggere il contenuto in posizione 1:' )
print ( 'Ovvero "( 3, 4 )".' )
print ( 'A questo punto bisogna immaginarsi come se ci fosse scritto "( 3, 4 ) [ 0 ]", ovvero le seconde [] indicano la posizione nella ennupla andata a leggere.' )
print ( )
print ( 'In altre parole i numeri sono ripettivamente 1 e 0, quindi bisogna andare a leggere prima in posizione 1, ed all\'interno della posizione 1, in posizione 0.' )
Output 
Se una ennupla contiene altre ennuple si può accedere ad i loro contenuti in questo modo:

3
In questo caso nelle prime [] vi è contenuto "1", questo indica che si debba leggere il contenuto in posizione 1:
Ovvero "( 3, 4 )".
A questo punto bisogna immaginarsi come se ci fosse scritto "( 3, 4 ) [ 0 ]", ovvero le seconde [] indicano la posizione nella ennupla andata a leggere.

In altre parole i numeri sono ripettivamente 1 e 0, quindi bisogna andare a leggere prima in posizione 1, ed all'interno della posizione 1, in posizione 0.
