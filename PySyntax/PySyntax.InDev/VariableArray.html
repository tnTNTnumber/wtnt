Variabili

Liste

Sintassi 

[ elemento numero 0, elemento numero 1, elemento numero 2, elemento numero 3, ... ]
Spiegazione: 
In Python, le liste, sono dei "contenitori" di elementi come variabili, numeri, stringhe, caratteri o booleani.

Esempio

Codice 
print ( [ 1, 2, 3 ] )
Output 
[1, 2, 3]

Esempio

Codice 
print ( [ 'a', 'b', 'c' ] )
Output 
['a', 'b', 'c']

Esempio

Codice 
X = 'Bistecca'
print ( [ 'Ciao', 5, True, X ] )
Output 
['Ciao', 5, True, 'Bistecca']

Esempio

Codice 
print ( [ [ 1, 2 ], [ [ 1, 2 ], 2, 3 ] ] )
Output 
[[1, 2], [[1, 2], 2, 3]]
