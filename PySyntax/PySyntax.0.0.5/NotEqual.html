Operatori Di Comparazione

Disegualianza

Sintassi 

primo termine != secondo termine
Output: 
True: Se i due termini sono diversi.
False: Se i due termini sono uguali.

Esempio

Codice 
print ( 5 != 5 )
Output 
False

Esempio

Codice 
print ( 5 != 3 )
Output 
True

Esempio

Codice 
print ( 'ciao' != 3 )
Output 
True

Esempio

Codice 
print ( 'ciao' != 'ciao' )
Output 
False

Esempio

Codice 
print ( 'gatto' != 'cane' )
Output 
True

Esempio

Codice 
print ( True != True )
Output 
False

Esempio

Codice 
print ( None != '' )
Output 
True

Esempio

Codice 
print ( '' != 0 )
Output 
True

Esempio

Codice 
print ( [ 1, 2, 3 ] != [ 1, 2, 3 ] )
Output 
False

Esempio

Codice 
print ( [ 1, 2, 3 ] != [ 'A', 'B', 'C' ] )
Output 
True

Esempio

Codice 
X = 5
print ( X != 5 )
Output 
False

Esempio

Codice 
X = 5
print ( X != X )
Output 
False

Esempio

Codice 
X = 5
Y = 3
print ( X != Y )
Output 
True
