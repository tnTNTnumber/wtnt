Variabili

Dichiarazione

Sintassi 

variable = contenuto
Spiegazione: 
In Python, la dichiarazione di una variabile, si può eseguire mediante un'assegnazione semplice.

Esempio

Codice 
X = 5
Y = True
Z = 'ciao'
W = [ 5, 'ghiande' ]
print ( X )
print ( Y )
print ( Z )
print ( W )
print ( '\nLe variabili possono contenere degli elementi, i più comuni sono:' )
print ( 'numeri, booleani, stringhe, array, ...' )
Output 
5
True
ciao
[5, 'ghiande']

Le variabili possono contenere degli elementi, i più comuni sono:
numeri, booleani, stringhe, array, ...
