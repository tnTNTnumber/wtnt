Dice Fight Game made in HTML, CSS and JavaScript (JS)

Dice Fight Game made in HTML, CSS and JavaScript (JS) is a game in which two dice fight, in a very random fight. Whoever rolls the highest number wins the battle. If both of them come up with the same number, he withdraws, and the same numbers that gave the tie cannot come out. If there is a tie with all 6 numbers, the battle ends in a tie. At the end of the challenge, whoever has scored the most points in the battles wins. If the total points are equal, no one wins. In the statistics the score of the matches is kept, it can go from -1 to 1. If you win a challenge you get +1, if you lose, instead, -1. The average of the points obtained gives the total scores. Furthermore, the statistics contain the number of games won, games lost, and played. The statistics are stored in the session of the browser.
